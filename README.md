# 📩 Spam Detector Project

A simple and effective machine learning-based web application that classifies messages as **Spam** or **Not Spam**.

---

## 🚀 Features

- 🔍 Detects whether a message is spam or not  
- ⚡ Real-time prediction using Flask API  
- 🎨 Clean and user-friendly UI  
- 🧠 Machine Learning model using Naive Bayes  
- 🌐 Frontend + Backend integrated  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Machine Learning:** Scikit-learn  
- **Libraries Used:**  
  - Pandas  
  - Scikit-learn  
  - CountVectorizer  
  - Multinomial Naive Bayes  

---

## 🧠 How It Works

1. User enters a message in the web interface  
2. The message is sent to the Flask backend  
3. The text is converted into numerical data  
4. A trained ML model predicts:
   - Spam 📛  
   - Not Spam ✅  
5. Result is displayed on the screen

## ▶️ How to Run This Project

### 1. Install dependencies

```
pip install flask flask-cors pandas scikit-learn
```

### 2. Run the Flask server

```
python app.py
```

### 3. Open the website

Open your browser and go to:
  http://127.0.0.1:5000
